About Me

I am a AWS Cloud Practitioner, who successfully completed the Cloud Resume Challenge, which involved creating and deploying a visitor-count enabled resume website on AWS.

More About Me

By completing the Cloud Resume Challenge, I gained invaluable, hands-on experience with an array of AWS cloud services, including S3, CloudFront, API Gateway, Lambda, Route 53, and Certificate Manager, all of which form the foundation of the website's infrastructure.

Utilizing tools such as Terraform, I was able to create a concise representation of the infrastructure in code form, known as Infrastructure as Code (IAC). Git, Github, and Github Actions allowed me to craft a robust CI/CD pipeline, which was a great milestone towards the project's success.

Of course, no challenge would be complete without encountering a few obstacles. I had the opportunity to address the notorious CORS (Cross-Origin Resource Sharing) error, learning valuable lessons along the way. Ultimately, the Cloud Resume Challenge provided me with an unmatched, immersive learning experience in the world of cloud computing.
